# Bay Car Service — Web Sitesi

Fethiye Taşyaka'da Alman grubu (Mercedes / BMW / Audi / VW / Porsche) özel servisi için kurumsal web sitesi.

## Stack

React 19 + Vite + TypeScript + Tailwind v4 + framer-motion + Lenis

## Geliştirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Üretim çıktısı `dist/` klasörüne yazılır. Önizleme için `npm run preview`.

## Notlar

- Randevu formu backend kullanmaz; gönderim WhatsApp deep-link ile iletilir.
- `og-image.svg`, harita koordinatı ve galeri/Hakkımızda görselleri geçici placeholder'dır; gerçek içerik geldiğinde güncellenecektir.
- `three` / `@react-three` paketleri ileride eklenecek 3D model entegrasyonu için tutulmaktadır.
